# Python – Exercícios do Curso em Vídeo
Repositório com exercícios resolvidos em Python, desenvolvidos durante meus estudos pelo Curso em Vídeo ministrado pelo professor Gustavo Guanabara.
Este material representa minha base de aprendizado em Python do mundo 1 ao mundo 4, com foco em lógica de programação, fundamentos da linguagem e boas práticas iniciais.

# Conteúdos abordados
- Variáveis e tipos de dados  
- Estruturas condicionais e de repetição  
- Listas, tuplas e dicionários  
- Funções e modularização  
- Tratamento de erros  
- Manipulação de arquivos
- Programação Orientada a Objetos (POO)

# Organização:
Os exercícios estão separados conforme os módulos do curso.

# Observação
Este repositório tem finalidade educacional.  
Os códigos refletem meu processo de aprendizado e poderão ser refatorados e melhorados no futuro conforme minha evolução em Python.
